HardWell - Tienda de Componentes 🖥️
¡Bienvenido a HardWell! Este es un proyecto interactivo que simula una tarjeta de producto para una memoria RAM de alto rendimiento. Permite a los usuarios gestionar la cantidad de productos y calcular el total a pagar en tiempo real.

🚀 Características
Cálculo en Tiempo Real: El total a pagar se actualiza automáticamente al sumar o restar productos.

Formato de Moneda: Los precios utilizan el formato de moneda chilena (es-CL) para una mejor lectura.

Diseño Futurista: Interfaz moderna con efectos de glassmorphism (desenfoque de fondo) y gradientes.

Controles Intuitivos: Botones para añadir, quitar y resetear (vaciar) la cantidad.

Responsivo: Adaptado para visualizarse correctamente en diferentes tamaños de pantalla.

🛠️ Tecnologías utilizadas
HTML5: Estructura semántica.

CSS3: Estilos personalizados, Flexbox y Google Fonts.

JavaScript (Vanilla): Lógica de manipulación del DOM y eventos.

Bootstrap 5: Soporte para la estructura base y navegación.

FontAwesome: Iconografía para los controles y el carrito.
